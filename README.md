# User Profile JSX Conversion

# Description
This project demonstrates converting standard HTML markup into JSX syntax for use in a React component. The UserProfile component accepts props and dynamically renders user information.

# Features
- Converts HTML to valid JSX
- Uses className instead of class
- Uses props for dynamic content
- Handles normal and edge cases

# How to Run

1. Clone the repository
2. Run: npm install
3. Run: npm start
4. Open http://localhost:3000

# Test Cases
Includes 3 normal test cases and 3 edge cases to verify functionality and error handling.
